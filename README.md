### ⚙️ Installation & Setup

**Prerequisites**
Make sure you have these installed:<br>

-Node.js v14+<br>
-npm or yarn<br>
-pnpm

--
1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/portfolio.git
    ```
2. Navigate to the project directory:
    ```bash
    cd portfolio
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
